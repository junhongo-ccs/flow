# AI見積もりシステム - ユーザーマニュアル

**対象者**: 営業部門・非技術者  
**最終更新**: 2025-12-30

---

## 📖 このシステムについて

AI見積もりシステムは、お客様からの開発案件の見積もりやデザイン相談に、AIが自動で回答するシステムです。

### できること

1. **開発見積もり**: プロジェクトの概要を入力すると、AIが見積もり金額を提示
2. **デザイン相談**: デザインフェーズの状況を入力すると、AIが適切なアドバイスを提供

---

## 📝 使い方

### 1. 開発見積もりの取得

お客様から開発案件の相談を受けた時に使用します。

#### 手順

1. **システムにアクセス**
   - URLを開く: https://zealous-river-0efdffa0f.1.azurestaticapps.net

2. **「開発見積もり」タブを選択**
   - 画面上部のタブが選択されていることを確認

3. **必要情報を入力**

   | 項目 | 説明 | 入力例 |
   |------|------|--------|
   | プロジェクトタイプ | 開発するシステムの種類 | Webアプリケーション |
   | 開発期間 | 予定している開発期間（月） | 6 |
   | チームサイズ | 開発チームの人数 | 3 |

4. **「見積もりを取得」ボタンをクリック**

5. **結果を確認**
   - 10〜20秒ほど待つと、AIが見積もり結果を表示します
   - 見積もり金額と詳細な内訳が表示されます

#### 入力例

```
プロジェクトタイプ: Webアプリケーション
開発期間: 6ヶ月
チームサイズ: 3名
```

#### 出力例

```
見積もり結果をお知らせします。

総計: ¥1,848,000

内訳:
- 基本開発費用: ¥1,680,000
- 画面単価: ¥120,000
- 難易度係数: 1.0（標準）
- バッファ係数: 1.1（リスク管理のため）

この見積もりは、14画面のWebアプリケーションを対象としており...
```

---

### 2. デザイン相談

お客様からデザインに関する相談を受けた時に使用します。

#### 手順

1. **「デザイン相談」タブを選択**
   - 画面上部のタブをクリック

2. **デザインフェーズの状況を入力**

   | 項目 | 説明 | 選択肢 |
   |------|------|--------|
   | ワイヤーフレーム作成済み | ワイヤーフレームが準備できているか | チェックボックス |
   | デザイン会社選定済み | デザイン会社が決まっているか | チェックボックス |
   | Figma経験 | Figmaの使用経験 | なし / 初心者 / 中級 / 上級 |
   | 画面数 | デザインする画面の数 | 数値入力 |
   | レスポンシブ対応必要 | スマホ対応が必要か | チェックボックス |

3. **「相談する」ボタンをクリック**

4. **結果を確認**
   - AIがデザインフェーズに関するアドバイスを提供します

#### 入力例

```
✓ ワイヤーフレーム作成済み
□ デザイン会社選定済み
Figma経験: なし
画面数: 20
✓ レスポンシブ対応必要
```

#### 出力例

```
デザインフェーズのアドバイス

ワイヤーフレームが準備されているのは素晴らしいスタートです。
デザイン会社の選定についてアドバイスします...

推奨事項:
- Figmaでのデザイン制作を推奨します
- デザイン費用の目安: ¥400,000 - ¥800,000
- 納期の目安: 4-6週間
```

**メリット**:

- お客様に具体的な提案ができる

## ⚠️ 注意事項

### 見積もりについて

> [!IMPORTANT]
> - この見積もりは**概算**です
> - 正式な見積もりは、詳細なヒアリング後に作成してください
> - 複雑な案件は、技術部門に相談することをお勧めします

### 個人情報について

> [!WARNING]
> - お客様の個人情報や機密情報は入力しないでください
> - プロジェクトの概要のみを入力してください

### システムの応答時間

- AIの回答には10〜20秒かかります
- インターネット接続が遅い場合、さらに時間がかかることがあります
- 「読み込み中」の表示が出ている間は、そのままお待ちください

## 🔧 トラブルシューティング

### 見積もりが表示されない

**原因**: インターネット接続の問題、またはシステムエラー

**対処法**:
1. インターネット接続を確認
2. ページを再読み込み（F5キー）
3. それでも解決しない場合は、システム管理者に連絡

---

### エラーメッセージが表示される

**表示例**: 「エラーが発生しました」

**対処法**:
1. 入力内容を確認（すべての項目が入力されているか）
2. ページを再読み込み
3. 再度入力して試す
4. 問題が続く場合は、システム管理者に連絡

---

### 画面が正しく表示されない

**対処法**:
1. ブラウザを最新版に更新
2. 別のブラウザで試す（Chrome、Edgeなど）
3. キャッシュをクリア

## 📚 よくある質問（FAQ）

### Q1: 見積もりはどのくらい正確ですか？

**A**: 概算見積もりです。実際のプロジェクトでは、詳細なヒアリングを行い、正式な見積もりを作成してください。目安として±20%程度の変動があります。

---

### Q2: 複数のプロジェクトタイプを組み合わせたい場合は？

**A**: それぞれのタイプで個別に見積もりを取得し、合算してください。または、技術部門に相談することをお勧めします。

---

### Q3: 見積もり結果を保存できますか？

**A**: 現在、システムには保存機能はありません。必要な場合は、結果をコピーしてWordやExcelに貼り付けてください。

---

### Q4: スマートフォンでも使えますか？

**A**: はい、スマートフォンでも使用できます。ただし、画面が小さいため、PCやタブレットでの使用を推奨します。

---

### Q5: オフラインでも使えますか？

**A**: いいえ、インターネット接続が必要です。
