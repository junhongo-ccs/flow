# AI見積もりシステム V2 ユーザーガイド

V2では、従来のフォーム形式からチャット形式の「デザイン・見積もり相談アシスタント」に進化しました。

## 主な特徴
- **対話型ヒアリング**: AIが対話を通じて要件を引き出します。
- **SIer基準の価格設定**: 社員800人規模の中堅SIer相場に基づいた精緻な見積もり。
- **デザイン重視**: UI/UXデザインフェーズの重要性を考慮した提案が可能です。
- **Markdown出力**: 相談完了後、詳細な提案書をMarkdown形式でダウンロードできます。

## アクセスURL
- **本番環境**: [https://zealous-river-0efdffa0f.1.azurestaticapps.net](https://zealous-river-0efdffa0f.1.azurestaticapps.net)
- **開発環境 (ローカル)**: [http://localhost:8080](http://localhost:8080) (または `index.html` を直接ブラウザで開く)

## 使い方
1. **チャットの開始**: 
   画面を開くとAIが挨拶します。最初に「Webアプリ」「モバイルアプリ」などのプロジェクトタイプを選択してください。
2. **ヒアリングへの回答**:
   AIの質問に対し、選択肢ボタンまたは自由記述で回答してください。
3. **詳細な質問**:
   「デザインにこだわりたい」「セキュリティを重視したい」などの要望を自由に伝えることができます。
4. **見積もりの生成**:
   必要な情報が揃うと、AIが「プロジェクト見積もり・提案書」を生成します。
5. **ダウンロード**:
   最後に表示される「見積もり書をダウンロード」ボタンをクリックして、提案書を保存してください。
6. **リスタート**:
   「最初からやり直す」ボタンをクリックすると、現在のセッションがリセットされ、最初から相談を始めることができます。

## 困ったときは
- **エラーが表示された場合**: 通信環境を確認し、ページをリロードして最初からやり直してください。
- **選択肢にない回答をしたい場合**: 「その他」を選択するか、テキスト入力欄に直接入力してください。
