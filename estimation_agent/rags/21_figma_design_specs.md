# Figmaデザイン仕様ガイド

## Figmaとは
ブラウザベースのデザインツール。デザイン会社との共有が容易で、開発者もコードを確認できる。

## デザイン会社から受け取るべきもの

### 1. Figmaファイル
- 全画面のデザイン
- コンポーネント化されたUI部品（ボタン、フォーム等）
- レスポンシブデザイン（PC/タブレット/スマホ）
- インタラクション・アニメーション仕様

### 2. デザインガイド（スタイルガイド）
- **カラーパレット**: プライマリ、セカンダリ、背景色等
- **タイポグラフィ**: フォント名、サイズ、行間、ウェイト
- **スペーシング**: 余白のルール（8pxグリッド等）
- **アイコンセット**: 使用するアイコン一覧
- **コンポーネント**: ボタン、フォーム、カード等の仕様

### 3. アセット
- 画像ファイル（PNG, SVG, WebP）
- アイコン（SVG推奨）
- ロゴ（各サイズ）
- ファビコン

## 開発者への引き継ぎ時のチェックリスト

- [ ] Figmaファイルへのアクセス権付与（閲覧権限）
- [ ] デザインガイドの共有（PDF）
- [ ] 使用フォントのライセンス確認
- [ ] 画像アセットの書き出し（適切な解像度）
- [ ] インタラクション・アニメーションの説明
- [ ] レスポンシブ対応の仕様（ブレークポイント）
- [ ] ホバー・フォーカス状態のデザイン
- [ ] エラー状態のデザイン

## Figmaの便利機能

### Dev Mode（開発者モード）
- CSS/React/iOS/Androidコードの自動生成
- スペーシング・カラーコードの確認が容易
- アセットの書き出し

### コメント機能
- デザイン上で直接フィードバック可能
- デザイン会社とのやりとりが効率化
- 修正履歴の追跡

### プロトタイプ機能
- 画面遷移のシミュレーション
- インタラクションの確認
- クライアントへのプレゼンに活用

## 費用目安（Figmaデザイン）

| 規模 | 画面数 | 費用 | 納期 |
|-----|-------|------|------|
| 小規模 | 5-10画面 | ¥200,000 - 400,000 | 2-3週間 |
| 中規模 | 10-20画面 | ¥400,000 - 800,000 | 4-6週間 |
| 大規模 | 20画面以上 | ¥800,000 - 1,500,000 | 6-8週間 |

**追加費用**:
- レスポンシブ対応（PC/タブレット/スマホ）: +30-50%
- アニメーション・インタラクション: +10-20%
- デザインシステム構築: +20-30%

## レスポンシブデザインの確認ポイント

### ブレークポイント
- **デスクトップ**: 1200px以上
- **タブレット**: 768px - 1199px
- **スマートフォン**: 767px以下

### 確認事項
- [ ] 各ブレークポイントでのレイアウト
- [ ] 画像の最適化（各サイズ）
- [ ] ナビゲーションの変化（ハンバーガーメニュー等）
- [ ] フォントサイズの調整
- [ ] タッチ操作の考慮（ボタンサイズ等）

## デザインシステムの重要性

### メリット
- 開発効率の向上（コンポーネント再利用）
- デザインの一貫性
- 保守性の向上
- スケーラビリティ

### 含まれるべき要素
- カラーパレット
- タイポグラフィ
- スペーシング
- コンポーネントライブラリ
